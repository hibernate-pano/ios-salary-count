# App Store 截图素材

模拟器 iPhone 17 Pro Max (6.9", 1320×2868) 生成。App Store 接受此尺寸作为 6.7"/6.9" 类截图。

## 文件
- `01_海蓝_收入页.png` —— 海蓝主题，实时收入主界面
- `02_樱粉_收入页.png` —— 樱粉主题
- `03_鎏金_收入页.png` —— 鎏金主题
- `04_深色_收入页.png` —— 深色模式 + 薄荷绿

## 用途
App Store Connect 要求至少 1 张 6.7"(或更大)截图。以上任选 2–4 张上传即可，建议都传，展示多彩主题卖点。

## 上传时的小建议
- App Store 允许在截图上叠加宣传文字（如"看着工资每秒往上涨""4 套主题随心换"）。如需带文字的精修版，可用任意图片工具在这些原图上加字，或告诉我用代码生成带标题的版本。
- 顺序建议：海蓝(主打) → 深色 → 樱粉 → 鎏金。

## 重新生成
切换主题/模式后用 `xcrun simctl io <device> screenshot` 截图即可。设备需为 Pro Max 类机型以满足尺寸要求。
